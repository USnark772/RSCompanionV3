To build app executable and package into installer:
-   Update version number and set release bool to True in app_defs.py
-   Update build path in comp_app_setup.py
-   Update version and build path in comp_app_inno.iss
-   run in console: python (path to comp_app_setup.py) build
        - example: "python C:\RSDev\asyncCompanion\For_setup\comp_app_setup.py build"
-   compile .iss using Inno Setup Compiler app
